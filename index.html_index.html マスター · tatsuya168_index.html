    ここではその中でもエネルギー機械コースについて紹介していきます。

    <br>・輸送機械や発電所等の産業基盤の設計と運用に必要な技術
    <br>・水素エネルギー等のクリーンエネルギーの創生と貯蔵に関する技術
    <br>・自然災害の予測分析と対策に関する技術
